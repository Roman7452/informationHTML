О HTML

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц. Он используется для структурирования информации на странице с помощью различных элементов, таких как заголовки, параграфы, списки и многое другое.

HTML состоит из так называемых тегов, которые определяют различные элементы на странице. Каждый тег заключается в угловые скобки <>, и обычно имеет открывающий и закрывающий теги, например <p> для параграфа.

О CSS

CSS (Cascading Style Sheets) - это язык таблиц стилей, который используется для стилизации HTML-документов. С помощью CSS вы можете изменять внешний вид элементов HTML, таких как цвет фона, шрифты, размеры и многое другое.

Чтобы применить стили к элементам HTML, вы можете использовать селекторы CSS. Селектор указывает, к каким элементам применяются стили, а декларация содержит стили, которые должны быть применены. Например:

        p {
            color: #333;
        }

В приведенном выше примере, стиль цвета (#333) применяется ко всем элементам <p> на странице.
